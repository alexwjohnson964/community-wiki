# Organization

Github Link: https://github.com/Code-after-Coffee

The previous link is the Github organization created to house all of the source repos for the Code after Coffee
programming community. This is where new projects for the community will begin and be managed for the benifit of 
anyone that is interested in working with us to learn, collaborate, and enhance their skills.

The community aims to provide a variaty of projects that members can contribute to across different skill levels 
and levels of experience.

## How to join

To join the organization and participate in contributing on currently ongoing projects simply join the Code after Coffee discord group by following this [link](https://google.com) if not already in the discord group and express your interest to join the Github organization to discord group admin alex (alexj1678)

## Repositories

Once a part of the organization members are able to interact with the existing repositories associated with the organization along with having the ability to create new ones in order to start new projects. Please ensure first that the project is mentioned in the discord group under the brainstorming ideas channel or in one of the skill domain relevant channels that relate to the proposed project in order to find people in the community that would be interested in the project.
